    Function floorDateToMinutes

    • floorDateToMinutes(everyMinutes?: number, date?: Date): Date

      Floors a given Date object down to the nearest time interval in minutes.

      Parameters

      • everyMinutes: number = 1

        The step interval (in minutes) used for rounding down. Must be a positive finite number.
        Values below 1 are treated as 1, values above 60 as 60.

      • date: Date = ...

        The input date to be floored.
        Defaults to the current time (new Date()).

      Returns Date

      A new Date instance, representing the same hour as the input, but with minutes rounded down to the nearest everyMinutes multiple.

      Remarks

      This utility function is commonly used for time bucketing, grouping logs, or aligning timestamps to fixed intervals (e.g. 5-minute or 15-minute marks).

      It takes an arbitrary Date and returns a new Date instance (it does not mutate the input) where:

      • The minutes value is floored down to the nearest multiple of everyMinutes.
      • seconds and milliseconds are reset to 0.
      • The hour and day remain unchanged.

      The step is automatically clamped to the range 1 – 60 minutes to prevent invalid or nonsensical values.

      The function allocates a single new Date object.
      It is safe for high-frequency use in real-time systems and event batching.

      Example

      // Example 1: Floor to the nearest 15-minute mark
      const d = new Date('2025-10-18T10:43:27');
      floorDateToMinutes(15, d);
      // -> 2025-10-18T10:30:00.000Z

      Example

      // Example 2: Using default date (current time)
      const nowFloored = floorDateToMinutes(10);
      // -> e.g. 2025-10-18T09:20:00.000Z

      Example

      // Example 3: Clamp behavior
      floorDateToMinutes(-5, new Date());  // treated as 1 minute
      floorDateToMinutes(999, new Date()); // treated as 60 minutes

      Throws

      Never throws — invalid step values are automatically normalized.

      See

      • Date#setMinutes for the underlying mutation logic.
      • Date#getMinutes for how minutes are extracted from a Date.
