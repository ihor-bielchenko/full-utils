    Function isFunc

    • isFunc<T extends Function = Function>(value: unknown): value is T

      Checks whether a given value is a function.

      Type Parameters

      • T extends Function = Function

        The expected function type to narrow to (defaults to the base Function type).

      Parameters

      • value: unknown

        Any value to check.

      Returns value is T

      true if the value is of type "function", otherwise false.

      Remarks

      Type narrowing

      When this function returns true, TypeScript infers that:

      value is T

      This allows safe calling of the function or using its specific signature.

      Behavior

      • Returns true for:
        • Normal functions: function test() {}
        • Arrow functions: () => {}
        • Async functions: async () => {}
        • Generator functions: function* gen() {}
        • Class constructors (typeof MyClass).
      • Returns false for:
        • Non-callable objects or primitives.
        • Function-like objects that aren't real functions (e.g., { call() {} }).

      Performance

      • Time complexity: O(1)
      • Space complexity: O(1)

      Examples

      Example

      // Basic usage
      isFunc(() => {});           // => true
      isFunc(function test() {}); // => true
      isFunc(class A {});         // => true
      isFunc(123);                // => false
      isFunc({});                 // => false

      Example

      // Type narrowing
      const fn: unknown = () => 'ok';
      if (isFunc<() => string>(fn)) {
        const result = fn(); // result: string
      }

      Example

      // Class constructors are also "functions"
      class MyClass {}
      isFunc(MyClass);            // => true
