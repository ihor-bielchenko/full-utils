    Interface PasswordOptions

    Configuration options that define password validation rules.

    This interface allows you to specify various constraints that determine whether a given password is considered valid.

    Example

    const options: PasswordOptions = {
      minLength: 8,
      maxLength: 32,
      requireUppercase: true,
      requireLowercase: true,
      requireDigit: true,
      requireSpecial: true,
    };

    Remarks

    You can use these options to build password validation functions, enforce strong password policies, or configure authentication modules.
